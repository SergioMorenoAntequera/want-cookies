# You want cookies...

Want cookies is a really small library to help you create custom cookies alert with 3 lines of code. You can customice every text to give the feel that you would like to give.

## Installation
### Getting the resouces
You will just need to include the next links:
``` 
    https://daefstgj.com/want-cookies/src/tc.css
    https://daefstgj.com/want-cookies/src/tc.js
```
